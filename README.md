# Student Record Management Web Application (Java)
A simple web application built using Java to manage student records. This project demonstrates how to use Java Servlets, JSP (JavaServer Pages), and MySQL to create, read, update, and delete student data in a web-based interface. The application allows users to manage student information, such as names, roll numbers, courses, and grades.
## Features:
1. CRUD Operations: Add, edit, and delete student records.
2. Database Integration: Stores student data in a MySQL database.
3. Web Interface: A simple front-end built using JSP for displaying student information and accepting input.
4. Servlets: Handles server-side logic for processing HTTP requests and database interactions.
## Technologies Used:
1. Java (JDK)
2. NetBeans IDE (for development)
3. Apache Tomcat (for deployment)
4. JSP & Servlets (for dynamic web pages)
5. MySQL (for database storage)
6. HTML/CSS (for basic user interface design
