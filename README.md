
Student Record Management Web Application (Java)
A simple web application built using Java to manage student records. This project demonstrates how to use Java Servlets, JSP (JavaServer Pages), and MySQL to create, read, update, and delete student data in a web-based interface. The application allows users to manage student information, such as names, roll numbers, courses, and grades.
Features:
•	CRUD Operations: Add, edit, and delete student records.
•	Database Integration: Stores student data in a MySQL database.
•	Web Interface: A simple front-end built using JSP for displaying student information and accepting input.
•	Servlets: Handles server-side logic for processing HTTP requests and database interactions.
Technologies Used:
•	Java (JDK)
•	NetBeans IDE (for development)
•	Apache Tomcat (for deployment)
•	JSP & Servlets (for dynamic web pages)
•	MySQL (for database storage)
•	HTML/CSS (for basic user interface design
